# HTML Structure Visualizer

An interactive HTML Structure Visualizer that turns raw HTML into colorful nested boxes. Each box represents an element in the DOM tree and includes tooltips with tag insights, attributes, and child counts — great for learning, teaching, and quickly understanding document structure.

---

## Features

- **Visual DOM tree**
  - Nested, color-coded boxes for elements (html, head, body, headings, paragraphs, lists, etc.)
  - Clear visual hierarchy that reflects parent–child relationships.
- **Rich tooltips on hover**
  - Short description of the HTML tag (e.g. “Block-level heading”, “Inline text container”).
  - Attributes and their values (id, class, href, src, etc.).
  - Number of child elements and whether it is self-closing.
- **Responsive layout**
  - Works on desktop, tablet, and mobile.
  - Scrollable and zoom-friendly layout for large documents.
- **Static and lightweight**
  - Pure front-end; no backend or build step required.
  - Works by simply opening index.html in the browser.
- **SEO-ready**
  - Includes basic meta tags and Open Graph tags (title, description, preview).

## Getting Started

### Option 1 – Open Locally (No Setup)

1. Download or clone this repository.
2. Open the file `index.html` directly in your browser (double-click or drag it into a browser window).
3. Start pasting HTML into the input area and click **Visualize**.
